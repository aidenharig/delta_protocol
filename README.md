# DELTA PROTOCOL

**A continuously evolving, science-based training system built for long-horizon athletic and tactical development.**

[![Current Protocol Preview](preview.html)](index.html)

Delta Protocol is a structured, self-coached program designed to develop a complete athlete from the ground up — one who is strong, capable, mobile, and conditioned for both physical performance and sustained output. The system is built on exercise science principles, RPE-based autoregulation, concurrent endurance training, and deliberate recovery infrastructure.

The goal is not a single training block. It is the accumulation of purposeful blocks over years — each one building on the last — progressing from a general fitness foundation toward elite hybrid athleticism and tactical physical capability.

---

## Philosophy

Every session, every phase, and every protocol decision in this system is grounded in the following:

- **Science over tradition.** Programming choices are justified by exercise physiology, not convention.
- **Autoregulation over prescription.** RPE governs load, not arbitrary percentages. The body's response to training is the data.
- **Recovery as training.** Sleep, mobility, nutrition, and mental state are programmed with the same rigor as lifting and conditioning.
- **Long horizon.** No block is optimized in isolation. Each phase is designed to create the conditions for the next one.

---

## System Architecture

```
Foundation → Hypertrophy → Strength Integration → Tactical Conditioning
```

The program advances through phases that build sequentially. Strength is not chased early. Muscle and work capacity are built first, creating the substrate that strength and tactical performance then map onto.

Concurrent endurance is managed throughout using an 80/20 polarized model to minimize interference with hypertrophy while steadily expanding aerobic base.

## Protocol Stack

Each block is built on a consistent infrastructure layer:

- **Core:** McGill Big 3 anti-movement circuit, progressive across each phase
- **Mobility:** FRC-based CARs, 10 min daily
- **Recovery:** 4x/week sauna protocol, post-session differentiated recovery
- **Nutrition:** Plate composition model, 1g/lb protein target, carb periodization
- **Hydration:** LMNT electrolyte protocol, day-type-specific targets
- **Monitoring:** Weekly self-audit, RPE tracking, bodyweight and bloodwork checkpoints

---

## Long-Horizon Goal

Build the physical and physiological foundation necessary to compete at the level of elite hybrid athletes and tactical operators — individuals defined not by a single quality but by the intersection of strength, endurance, mobility, and mental resilience.

Each block is one step in that direction.
